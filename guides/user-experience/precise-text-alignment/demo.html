Precise Text Alignment

When rendering multiline text, the "ghost space" above the first line's cap height and below the last line's baseline can make the container's physical padding look inaccurate. text-box: trim-both cap alphabetic; trims this internal leading off the block edges of the paragraph, ensuring your actual spacing matches your intended CSS padding.

(The striped magenta areas represent exactly 32px of CSS padding from the top and bottom of the card.)

Ghost Space (Untrimmed)

Many modern display fonts have large internal leading. Without trimming, the text's bounding box includes this invisible space above and below the visible letters. As you can see, the distance to the actual letters looks much larger than 32px. 

/* Container has exactly 32px padding,
   but the visual gap is much larger. */
.card {
  padding: 32px;
}

.card p {
  font-family: 'Space Grotesk';
}

Precise Fit (Trimmed)

Using text-box-trim removes the ghost space. The padding now perfectly hugs the top of the capital letters and the bottom of the alphabetic baseline. The visual gap matches your CSS values exactly. 

/* The paragraph's block edges are trimmed,
   so the 32px gap is perfectly accurate. */
.card {
  padding: 32px;
}

.card p {
  font-family: 'Space Grotesk';
  text-box: trim-both cap alphabetic;
}

Icon Alignment

When aligning text next to an icon using flexbox, the text's internal leading will often prevent perfect visual centering. Trimming the text-box allows layout engines like Flexbox or Grid to align against the actual visible letters.

(The dashed borders show the actual bounding boxes used for flex alignment.)

Misaligned (Untrimmed)

Info

/* The text box includes ghost space,
   skewing the flex center alignment. */
.icon-card {
  display: flex;
  align-items: center;
}

.icon-text {
  font-family: 'Space Grotesk';
}

Perfect Center (Trimmed)

Info

/* The block edges hug the letters, so
   flex container perfectly centers it. */
.icon-card {
  display: flex;
  align-items: center;
}

.icon-text {
  font-family: 'Space Grotesk';
  text-box: trim-both cap alphabetic;
}

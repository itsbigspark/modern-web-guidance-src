Animated Scrollbar Colors

A progressive enhancement using CSS @property and Scroll-Driven Animations.
Scroll down slowly inside the box.

Top (Pink)

Scroll down. As the container scrolls, the embedded CSS variable bound by animation-timeline: scroll(self) begins interpolating to Orange.

Middle (Orange)

The standard scrollbar-color property is technically non-animatable, but passing in an interpolated @property bypasses that limitation entirely, yielding a gorgeous color transition without a drop of JavaScript.

Bottom (Cyan)

You have reached the bottom! The CSS scroll timeline hits 100%, stabilizing the thumb color to a deep Cyan.

- The `body` element must have a `background-image` that contains a `linear-gradient` generated by a CSS `@function`.
- All `h2` elements must have a `background-image` that contains a `linear-gradient` generated by a CSS `@function`.
- When the viewport is 400px wide, the `main` element's `grid-template-columns` must use a function to resolve to a single column (e.g., `1fr`).
- When the viewport is 1000px wide, the `main` element's `grid-template-columns` must use a function to resolve to two columns of equal width.
- All uses of functions must be preceded with a fallback value for the same property.

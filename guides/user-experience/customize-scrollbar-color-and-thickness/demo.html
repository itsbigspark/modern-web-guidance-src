Custom Color & Thin

This box demonstrates a scrollable container with customized scrollbar color AND thickness.

In browsers that support the standard properties, `scrollbar-color` and `scrollbar-width: thin` are used.

In browsers that only support legacy vendor prefixes, `::-webkit-scrollbar` overrides both size and color within an isolating feature query.

No Scrollbar

This box demonstrates a scrollable container that has its scrollbar completely hidden, but remains scrollable!

In browsers that support standard properties, `scrollbar-width: none` is used.

In browsers that only support legacy webkit prefixes, `display: none` is applied directly to `::-webkit-scrollbar`.

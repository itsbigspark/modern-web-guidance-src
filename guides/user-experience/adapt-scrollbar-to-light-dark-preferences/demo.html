Custom Adaptive

This box has custom explicitly defined colors for its scrollbar.

It uses a @media (prefers-color-scheme: dark) CSS media feature to override the explicit light colors with darker ones.

This ensures custom aesthetics remain legible and visually appealing in both contexts.

Trial Expiration Calculator

Calculate subscription trial metrics mathematically using Temporal.ZonedDateTime. Use .since() to find elapsed active time and .until() for remaining time, demonstrating accurate, balanced duration objects that handle month boundaries correctly.

🚫
This demo requires a browser with native Temporal support

Code Running
// 1. Get current time point and zone
const now = Temporal.Now.zonedDateTimeISO();
const tz = now.timeZoneId;

// 2. Parse inputs
const start = Temporal.PlainDateTime.from("...").toZonedDateTime(tz);
const end = Temporal.PlainDateTime.from("...").toZonedDateTime(tz);

// 3. Calculate with .since() and .until()
const timeActive = now.since(start, { largestUnit: 'year' });
const timeRemaining = now.until(end, { largestUnit: 'year' });

High Contrast Scrollbar Adaptation

Scroll down inside the box. Use Chrome DevTools (Rendering tab) to emulate the prefers-contrast: more media feature and watch the scrollbar switch to a highly legible black-and-white theme.

This container features a scrollbar that is intentionally subtle by default, utilizing light gray tones for the track and thumb.

However, accessibility is crucial. Users with visual impairments may enable high contrast modes in their operating system to improve legibility.

When high contrast is detected via @media (prefers-contrast: more), the CSS custom properties defining this scrollbar will automatically swap.

The subtle gray tones are instantly replaced with stark black and white, guaranteeing the scrollbar thumb is clearly demarcated against the track and page background.

This demonstrates progressive enhancement for accessibility without sacrificing default aesthetics.

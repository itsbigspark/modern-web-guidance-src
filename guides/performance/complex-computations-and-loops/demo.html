Complex Computations and Loops

This demo illustrates how to prevent long-running JavaScript tasks from freezing the user interface. When a heavy computation runs synchronously, it blocks the main thread, stopping layout and paint animations and making clicks unresponsive.

The Task: We simulate heavy work by running 1000 iterations of a loop that performs 1,000,000 complex math operations (sine, cosine, sqrt) per iteration.

Test responsiveness: Click the "Click Me" button continuously while running either task.

Should freeze during blocking task, spin during yielding task.

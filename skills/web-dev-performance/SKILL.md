# Web Dev Performance

This skill provides guides for optimizing web application performance using modern APIs and techniques. Review the following descriptions and read the guides if they are relevant to the request.

## Available Guides

- **Adaptive Loading**: Load a fallback image when network conditions are poor using the Adaptive Loading API. See [guides/adaptive-loading.md](guides/adaptive-loading.md).
- **Break Long Tasks**: Improve interaction responsiveness (INP) by yielding to the main thread. See [guides/break-long-tasks.md](guides/break-long-tasks.md).
- **Lazy Load Images**: Defer loading of off-screen images to minimize network contention and improve LCP. See [guides/lazy-load-images.md](guides/lazy-load-images.md).
- **Preload & Prerender**: Improve navigation speed by preloading key resources or prerendering pages before the user clicks. See [guides/preload-prerender.md](guides/preload-prerender.md).
- **Content Visibility**: Improve initial page load and interaction responsiveness by deferring the rendering work of off-screen components until they are needed. See [guides/content-vis.md](guides/content-vis.md).

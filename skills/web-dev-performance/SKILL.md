# Web Dev Performance

This skill provides guides for optimizing web application performance using modern APIs and techniques.

## Available Guides

- **Adaptive Loading**: Load content based on network conditions. See [guides/adaptive-loading.md](guides/adaptive-loading.md).
- **Break Long Tasks**: Use the Scheduler API to keep the main thread responsive. See [guides/break-long-tasks.md](guides/break-long-tasks.md).
- **Lazy Load Images**: Defer loading of off-screen images. See [guides/lazy-load-images.md](guides/lazy-load-images.md).
- **Preload & Prerender**: Use Speculation Rules to prefetch content. See [guides/preload-prerender.md](guides/preload-prerender.md).

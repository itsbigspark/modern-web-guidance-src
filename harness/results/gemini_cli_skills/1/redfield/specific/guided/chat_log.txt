I have successfully modernized the tooltip implementation in `index.html`.

Here's a summary of the changes:
1.  **Removed Legacy JavaScript:** Eliminated the manual `mouseover`/`mouseout` event listeners and imperative DOM manipulation.
2.  **Implemented Popover API:** Added the `popover="hint"` attribute to the tooltip element, designating it as a lightweight, non-modal popover.
3.  **Added Interest Invokers:** Used the `interestfor` attribute on the button to declaratively link it to the tooltip, enabling hover and focus behavior without custom JavaScript.
4.  **Adopted CSS Anchor Positioning:** Replaced absolute positioning hacks with `anchor-name` on the button and `position-anchor` / `position-area` on the tooltip for robust, declarative positioning.
5.  **Included Polyfills:** Added script tags for `@oddbird/popover-polyfill`, `@oddbird/css-anchor-positioning`, and `interestfor` to ensure compatibility across browsers while these features are stabilizing.
6.  **Updated Styling:** Refined the CSS to work with the new positioning model, including `@starting-style` for smooth entry animations.

I also created `resources_used.json` to document the resources utilized during this task.

I have built the product landing page for "The Daily Grind" coffee shop in `index.html`.

This implementation features:
1.  **Scroll-Driven Animation:** The hero image fades in and grows as you scroll down, using the modern CSS `animation-timeline: view()` property (with a fallback for reduced motion preferences).
2.  **Tooltip with `interestfor`:** The "Details" button uses the new `interestfor` attribute and Popover API to show the ingredients tooltip on hover.
3.  **Adaptive Loading:** The main hero image includes a `loading-placeholder` attribute. A lightweight script mimics the behavior of loading a low-quality placeholder image (simulated with a blurred, low-res version) if the user is on a slow network (2G/3G).
4.  **Stock Images:** High-quality Unsplash images are used for the hero background.
5.  **Polyfills:** I have included the necessary polyfills (Interest Invokers, Popover, CSS Anchor Positioning) from CDNs to ensuring cross-browser compatibility where possible.

I also created `resources_used.json` to track the modern web guides used.

You can verify the page by opening `index.html` in a browser.

**Note:** The "adaptive loading" behavior is simulated with a script because it is currently an experimental feature. The script checks `navigator.connection` and swaps the image source if a slow connection is detected.
